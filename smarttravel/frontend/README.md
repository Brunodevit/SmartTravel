# Frontend - SmartTravel

Este diretório contém o frontend da aplicação SmartTravel, desenvolvido com Vue 3, Vite e TypeScript.

---

## 📌 Objetivo

Responsável pela interface do usuário da aplicação SmartTravel, incluindo:
- páginas (views)
- componentes reutilizáveis
- navegação (router)
- consumo de API futura

---

## 🛠️ Tecnologias

- Vue 3
- Vite
- TypeScript
- Vue Router (futuro)
- Axios (futuro)

---

## 🚀 Como executar

```bash
npm install
npm run dev
