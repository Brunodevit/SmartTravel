# Projeto da Solução

## 🧪 Tecnologias Utilizadas

O projeto **SmartTravel** foi inicialmente desenvolvido como uma aplicação web utilizando tecnologias fundamentais do desenvolvimento front-end, com foco em estruturação, consumo de dados simulados e interação com o usuário.

### 🖥️ Frontend (Estado inicial do projeto)

- HTML5 (estrutura das páginas)
- CSS3 (estilização e layout)
- JavaScript (lógica da aplicação e manipulação do DOM)
- Bootstrap (responsividade e componentes prontos)
- Font Awesome (ícones)
- JSON Server (simulação de API REST para dados locais)

---

### 📦 Simulação de Backend (fase inicial)

- JSON Server
  - Simulação de banco de dados com arquivo `db.json`
  - CRUD de usuários, roteiros e recomendações
  - Uso durante desenvolvimento sem backend real

---

### 🧠 Ferramentas de Desenvolvimento

- Visual Studio Code (IDE)
- Live Server (execução local do frontend)
- Git / GitHub (versionamento de código)
- Navegadores (Chrome / Firefox para testes)

---

### 🔄 Fluxo da Aplicação (User Flow)

1. Usuário acessa a aplicação via navegador
2. Página inicial (HTML + CSS + JS)
3. Interação com filtros e busca de destinos
4. Requisições para o JSON Server (API simulada)
5. Retorno de dados (roteiros, avaliações, recomendações)
6. Renderização dinâmica no frontend
7. Salvamento de dados locais (ex: sessão do usuário)
